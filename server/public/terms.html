利用規約

最終更新日: 2026年3月22日

本利用規約(以下「本規約」)は すれちがいライト(以下「本アプリ」)の利用条件を定めるものです. 本アプリを利用する全てのユーザーは 本規約に同意したものとみなします.

第1条 定義

  • 「本アプリ」とは すれちがいライト及び関連するサーバー・サービスの全体を指します
  • 「ユーザー」とは 本アプリを利用する全ての個人を指します
  • 「すれ違い」とは 本アプリのユーザー同士がGPS位置情報に基づき近くにいたと判定されることを指します
  • 「ひとこと」とは すれ違い時に他のユーザーに共有される短いテキストメッセージを指します
  • 「運営者」とは 本アプリの開発・運営を行う者を指します

第2条 アカウントと認証

  1. 本アプリはゲスト認証(端末UUID)により 登録手続きなしで利用できます
  2. Apple ID・Googleアカウントによるログインは任意です
  3. ユーザーは自身のアカウント情報を適切に管理する責任を負います
  4. 端末の紛失・変更等によりゲストアカウントのデータが失われた場合 運営者は復旧の義務を負いません

第3条 位置情報の利用

  1. 本アプリはすれ違い判定のためにGPS位置情報を取得します
  2. バックグラウンドでの位置情報取得を含みます
  3. 位置情報の利用を許可しない場合 本アプリの主要機能を利用できません
  4. 位置情報の取り扱いについては プライバシーポリシーをご確認ください

第4条 禁止事項

ユーザーは以下の行為を行ってはなりません.

  • 他のユーザーへの嫌がらせ・脅迫・誹謗中傷
  • わいせつ・暴力的・差別的な内容のひとこと・ニックネームの設定
  • 虚偽の位置情報を送信する行為(GPSスプーフィングなど)
  • 本アプリのサーバーやシステムに不正にアクセスする行為
  • APIの不正利用・リバースエンジニアリング・スクレイピング
  • 他のユーザーの個人情報を収集・蓄積する行為
  • 商業目的での本アプリの利用(運営者の許可なく)
  • 法令または公序良俗に反する行為
  • その他 運営者が不適切と判断する行為

第5条 コンテンツについて

  1. ユーザーが設定するニックネーム・ひとこと・アバターについて ユーザー自身が全ての責任を負います
  2. 運営者は 不適切と判断したコンテンツを予告なく削除できます
  3. ユーザーが投稿したコンテンツの著作権はユーザーに帰属します
  4. ユーザーは 本アプリ内でコンテンツが表示されることに同意します

第6条 通報・ブロック機能

  1. ユーザーは不適切なユーザーを通報・ブロックすることができます
  2. 通報内容に基づき 運営者は該当ユーザーへの警告・利用停止等の措置を取ることがあります
  3. ブロックしたユーザーとのすれ違いは記録されません

第7条 サービスの変更・中断・終了

  1. 運営者は事前の通知なく本アプリの内容を変更できます
  2. サーバーメンテナンス・障害・天災等により 本アプリが一時的に利用できない場合があります
  3. 運営者は 一定の予告期間をもって本アプリのサービスを終了できます
  4. サービスの変更・中断・終了により生じた損害について 運営者は一切の責任を負いません

第8条 広告の表示

本アプリは将来的に広告(Google AdMob等)を表示する場合があります. 広告の表示に関連して第三者サービスがユーザーの情報を取得する場合があります. 詳細はプライバシーポリシーをご確認ください.

第9条 免責事項

  1. 本アプリは「現状のまま」で提供され 運営者は明示・黙示を問わず一切の保証をしません
  2. すれ違い判定の正確性について 運営者は保証しません
  3. ユーザー間のトラブルについて 運営者は一切の責任を負いません
  4. 本アプリの利用により生じた損害(データの喪失を含む)について 運営者は故意または重大な過失がある場合を除き責任を負いません
  5. 位置情報の取得に伴うバッテリー消費について 運営者は責任を負いません

第10条 アカウントの停止・削除

  1. 運営者は 本規約に違反したユーザーのアカウントを予告なく停止・削除できます
  2. ユーザーは アプリ内の設定画面またはメールでの連絡により 自らのアカウントを削除できます
  3. アカウント削除後 ユーザーのデータは合理的な期間内に全て削除されます

第11条 知的財産権

本アプリに関する著作権・商標権その他の知的財産権は 運営者またはライセンス元に帰属します. ユーザーは 運営者の許可なく本アプリのコンテンツを複製・転載・改変することはできません.

第12条 規約の変更

運営者は必要に応じて本規約を変更することがあります. 重要な変更がある場合はアプリ内で通知します. 変更後も本アプリを継続して利用する場合 変更後の規約に同意したものとみなします.

第13条 準拠法と管轄

本規約は日本法に準拠します. 本アプリに関する紛争については 東京地方裁判所を第一審の専属的合意管轄裁判所とします.
